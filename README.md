WiiM is a Raycast extension to control WiiM devices.

## Installation 🛠️

To install the WiiM extension, follow these steps:

1. Open Raycast.
2. Search for "Store" and navigate to the Raycast Store.
3. Search for "WiiM" and click "Install."

## Usage 🚀

Once installed, simply trigger the Raycast command palette and search for the WiiM commands.

<p align="center">
  <img src="metadata/wii-m-1.png" alt="Raycast Command Palette for WiiM">
</p>

## Features ✨

### 1. `Play/Pause`

This command will toggle the playback state of the WiiM device.

### 2. `Next Track`

This command will skip to the next track on the WiiM device.

### 3. `Previous Track`

This command will skip to the previous track on the WiiM device.

### 4. `Set Volume`

This command will set the volume of the WiiM device to a specified level.
You can enter a value between `0` and `100` to set the volume.

### 5. `Volume Up`

This command will increase the volume of the WiiM device by a specified step.

### 6. `Volume Down`

This command will decrease the volume of the WiiM device by a specified step.

## Preferences ⚙️

### 1. `IP Address`

You can set the default IP address to use when connecting to a WiiM device. The default IP address is `192.168.1.100`.

### 2. `Volume Step`

You can set the default volume step to use when adjusting the volume on a WiiM device. The default volume step is `5`.
